La Coercition

Le moteur JavaScript change automatiquement le type des données pour essayer de faire l'opération :


5 + null    // retourne 5         null est convertit en 0
"5" + null  // retourne "5null"   null est convertit en "null"
"5" + 2     // retourne 52        2 est convertit en "2"
"5" - 2     // retourne 3         "5" est convertit en 5
"5" * "2"   // retourne 10        "5" et "2" sont convertit en 5 et 2

Les objets

Une entité qui possède des properiétés (variables) qui lui sont attachées.


// syntaxe litérale
var voiture = {couleur: "route"};    

// création basée sur un prototype
var voiture2 = Object.create(voiture); 

// définition d'un constructeur
function Moto(couleur) {this.couleur = couleur;} 
// création d'une nouvelle instance.
var motoRouge = new Moto('rouge');

Les prototypes

Le prototype est un objet partagé par toutes les instances du constructeur


function Moto(couleur) {this.couleur = couleur;} 

var motoRouge = new Moto('rouge'); 

console.log(motoRouge.type); // undefined

Moto.prototype.type = 'sportive';

var motoBleu = new Moto('bleu');
motoBleu.type = 'roadster';

console.log(motoRouge.type); // "sportive"
console.log(motoBleu.type); // "roadster"
